CSS position values

1 2 3 4 5

position: static

Es el valor por predeterminado, no se necesita escribir explicitamente. Se usa cuando no se necesita mostrar el contenido de ninguna manera especial, así de simple.

position: relative

Para posicionar relativo a su posición normal, donde estaría con static. Se usan las propiedades de left, top, right y bottom para dar el offset que quieras relativo a la posición normal. El demás contenido lo trata como si siguiera donde mismo. Un caso de uso muy común es cuando quieres que un position absolute se posicione en base a su padre, entonces al padre le pones relative.

position: absolute

Se posiciona relativo al ancestro más cercano que tenga una position diferente a static. Si no existe este ancestro, es relativo al body. Igualmente se usa left, top, etc. A diferencia de relative, demás contenido actua como si ya no existeria y pueden tomar su posición, por esto mismo un caso de uso es cuando quieres poner una cosa sobre la otra.

position: fixed

Se posiciona relativo a la ventana del navegador, por ejemplo, siempre top-right, siempre bottom-left, etc. Ejemplo muy común es un header que se queda arriba incluso cuando haces scroll hacia abajo.

position: sticky

Es un híbrido entre relative y fixed. Es relativo hasta que se llega al offset especificado. Caso de uso común es cuando se tienen multiples headings, y de acorde a que se hacia scroll down, los headings se van pegando arriba uno tras el otro.
